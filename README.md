## üìö Description

This repository serves as a **solution bank** for the course **"Algorithms for DNA Sequencing"**. It includes:  

- Homework solutions for programming assignments  
- Sample genome datasets for practicing DNA sequence analysis  
- Implementations of sequence alignment algorithms  
- Data structures and algorithms relevant for DNA sequencing tasks  

The goal is to provide learners with a **reference and learning aid** to understand how algorithms are applied in genomic data science and to practice coding solutions for real DNA sequencing problems.
